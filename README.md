# MIMO Systems Aided by Microwave Linear Analog Computers: Capacity-Achieving Architectures with Reduced Circuit Complexity

This repository contains code to reproduce the paper:

> M. Nerini, B. Clerckx,
> "[MIMO systems aided by microwave linear analog computers: Capacity-achieving architectures with reduced circuit complexity](https://ieeexplore.ieee.org/document/),"
> IEEE Trans. Wireless Commun., 2026.

## Code

The script `main_Fig7.m` reproduces Fig. 7(a) with `SNR_dB = 0` and Fig. 7(b) with `SNR_dB = 10`.

The script `main_Fig8.m` reproduces Fig. 8.
